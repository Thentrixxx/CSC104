CSS History and Information

CSS Definition

CSS stands for Cascading Style Sheets. CSS makes the HTML webpages look good. CSS can control basically any element on the page can be changed from the default style with CSS.

History of CSS

  • CSS1 was released in 1996
  • CSS2 was released in 1998
  • CSS3 is the current version, and is a living standard.

Basic Structure of CSS

selector{ property: value; property2: value; }

  • CSS selectors can be an HTML element (example p, body), id of an HTML element, custom class
  • CSS helps with the separation of content and looks
  • Easy to maintain
  • Accessibility issues can be more easily maintained

There are three ways to inculde CSS on a webpage

  • External - this is the best practice - CSS is in a CSS file
  • Embedded - This applies to one page.
  • Inline - This applies to a specific HTML element
